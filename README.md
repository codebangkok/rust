#### Rust Installation
Linux or macOS
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
Windows
```
https://static.rust-lang.org/rustup/dist/i686-pc-windows-gnu/rustup-init.exe
```

#### Fix Cargo Expand
```
rustup install nightly
```
